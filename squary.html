Introduction

Welcome to Squary.js,

The little Free JavaScript library that will help you making Isotope-like grid system.

It can be use for news system, events or any list item you want. It's easy to use, fast to setup and customizable.

Features & Limitations

Advantages

  • Flexible & Customizable
  • Easy to setup & use
  • Super Fast & Super Light
  • Filtering options
  • Totally responsive
  • Compatibility : IE9+ / Chrome / Firefox / Safari
  • Completly Free & Open Source

Limitations

  • Still in developement, early version
  • Basic, no advanced usage
  • Only on modern browsers

Installation

Download the minified version or the dev version.

Include your squary.js | squary.min.js at the bottom of your html file like following :

<script type="text/javascript" src="squary.min.js"></script>
<script type="text/javascript" src="yourcode.js"></script>

And that's all !

Dev Note : The bower package is coming.

Getting started

To setup your Squary Grid, you will need to have a container which will be your Grid and children which will be your Elements.

You will define the container as a Squary Grid and then Squary.js will automatically take all his children as block of the grid.

<section class="my-container">
    <article class="child"></article>
    <article class="child"></article>
    <article class="child"></article>
</section>

Alright for the HTML part, then you need to setup with JavaScript like following :

var squaryGrid = Squary.new(
    ".my-container", //   Container class
    6,             //   Resolution
    false,          //   Randomize
    0.3,           //    Transition time
    true,          //    Pixel Correction
    function(){}     //   Callback
);

Let's see what parameters are here and how you can customize your grid :

  • Container class : Put the selector (only one element) of the container, every child will be elements of the grid, little children are not affected.
  • Resolution : This is the density of the grid based on the width of the container. More upper value means more blocks per line. Technically blocks will always have almost the same size.
  • Randomize : Shuffle all children if true.
  • Transition time : The transition time for all animations. 0 means no animations.
  • Pixel Correction : Try with and without, it will add 1 pixel to the width and the height of each block to reduce gap between them.
  • Callback : Function that will be called at the end of the initialisation of the grid.

Of course you can setup multiple grid in your page.

Note : Each block has a width and a height setup with JavaScript and there is no overflow:hidden by default, so be careful with the content.

At this moment, your grid will already work !

Attributes

But did you notice ? Every block has the same 1x1 size ! Boring...

This is where attributes comes ! They need to be place on the child (here our article element)

  • squary-size="2x3" : You can setup the size of a block like : 1x2, 2x1, 3x1, 3x2, 4x2 ... you get it. By default and if this attribute is not existing, every blocks will be 1x1.
  • squary-filter="something" : It will add a filter to the block, then you can easily filter them with the method 
    .filter()
    described below.
  • squary-hidden="true" : It will completely hide the block.

Methods

Let's see now, how we can interact with all this.

The method are called on an Squary instance only. Create one like this : 

var calledLikeYouWant = Squary.new(parameters);

  • .new(String container,
            int resolution,
            bool randomize,
            int transitionTime,
            bool pixelCorrection,
            function callback)
     : The method that initialize the grid, should be use only once on a Squary instance. See upper for the parameters.
  • .refresh()
     : It will refresh the grid. In case you modify the DOM of the grid's elements or resize the container.
  • .onClick(function something)
     : Add a function that is called when a click is triggered on a block. It will be the same function for every block.
  • .filter(String aFilter)
     : It will keep only the block with the attribute squary-filter="aFilter" It will refresh the grid with the new elements. Call .filter("all") to show every elements.
  • .getBlock(int i)
     : Return the block element with the index i. Start at 0.
  • .setResolution(int i)
     : Modify the grid resolution and then refresh.
  • .setTransitionTime(int i)
     : Modify the transition time of animations and then refresh.

Note : All those methods are chainable.
